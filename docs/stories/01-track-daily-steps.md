# User Story: 1 - Track Daily Steps

**As a** health-conscious user,
**I want** to track my daily steps automatically,
**so that** I can monitor my physical activity and work towards my fitness goals.

## Acceptance Criteria

* Steps are automatically tracked using HealthKit integration
* Daily step count is displayed prominently on the dashboard
* Progress towards daily goal (8,000-10,000 steps) is shown with a visual progress circle
* Historical step data is preserved and accessible

## Notes

* Integrates with Core Motion and HealthKit APIs
* Target range: 8,000-10,000 steps daily as recommended by health experts
* Forms part of the core MVP functionality
