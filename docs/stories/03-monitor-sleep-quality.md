# User Story: 3 - Monitor Sleep Quality

**As a** health-conscious user,
**I want** to track my sleep duration and quality,
**so that** I can understand my sleep patterns and improve my rest.

## Acceptance Criteria

* Sleep data is automatically captured via HealthKit integration
* Sleep duration is displayed on the dashboard
* Progress towards 7-9 hour sleep goal is visualized
* Sleep quality metrics are shown when available

## Notes

* Target range: 7-9 hours of sleep per night
* Integrates with HealthKit for automatic sleep tracking
* Will be added after MVP core features (steps and water)
