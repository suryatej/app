# User Story: 5 - Receive Movement Reminders

**As a** health-conscious user,
**I want** to receive gentle reminders to move when I've been inactive,
**so that** I can break up prolonged sitting periods and maintain better health.

## Acceptance Criteria

* Push notifications sent after 30-60 minutes of detected inactivity
* Reminders are gentle and motivational, not overwhelming
* Core Motion API used to detect activity/inactivity
* Users can customize reminder frequency and timing

## Notes

* Movement breaks recommended every hour for optimal health
* Uses Core Motion for activity detection
* Notifications should be encouraging, not punitive
