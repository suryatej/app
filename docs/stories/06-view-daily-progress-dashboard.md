# User Story: 6 - View Daily Progress Dashboard

**As a** health-conscious user,
**I want** to see all my daily wellness progress in one unified dashboard,
**so that** I can quickly understand my overall health status and achievements.

## Acceptance Criteria

* Dashboard displays progress circles for each tracked metric (steps, water, sleep, meditation)
* Daily progress is clearly visualized and easy to understand
* Quick access to log activities (especially water intake)
* Clean, uncluttered interface focusing on key metrics

## Notes

* Central hub for all wellness tracking
* Should provide at-a-glance view of daily progress
* Core component of user experience design
