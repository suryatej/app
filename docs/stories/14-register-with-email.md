# User Story: 14 - Register with Email

**As a** new user,
**I want** to register for an account using my email address and password,
**so that** I can create a personal profile and start tracking my wellness journey.

## Acceptance Criteria

* Registration form includes fields for email, password, and password confirmation
* Email validation ensures proper format
* Password requirements are clearly displayed (minimum length, complexity)
* Successful registration creates a user account and logs the user in
* Error messages are clear and helpful for validation failures
* Terms of service and privacy policy links are provided

## Notes

* Part of authentication flow for onboarding new users
* Should provide clear feedback on password strength
* Consider email verification for security
