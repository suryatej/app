# User Story: 15 - Sign In with Email

**As a** registered user,
**I want** to sign in using my email address and password,
**so that** I can access my wellness data and continue tracking my health goals.

## Acceptance Criteria

* Login form includes email and password fields
* Successful authentication grants access to the user's dashboard
* Invalid credentials display a clear error message
* "Forgot password" option is available
* Session persists appropriately based on user preference (remember me option)

## Notes

* Essential for returning users to access their data
* Should provide secure authentication
* Consider biometric authentication for enhanced user experience
