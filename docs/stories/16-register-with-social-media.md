# User Story: 16 - Register with Social Media

**As a** new user,
**I want** to register using my social media accounts (Google, Apple, Facebook),
**so that** I can quickly create an account without filling out lengthy forms.

## Acceptance Criteria

* Social media registration options are clearly displayed (Google, Apple, Facebook)
* Selecting a social provider initiates OAuth authentication flow
* Successful authentication creates a user account with profile information from social provider
* User can access the app immediately after social registration
* Privacy notice explains what data is accessed from social accounts

## Notes

* Reduces friction in onboarding process
* Target audience (millennials and Gen Z) prefer social sign-in options
* Must comply with each platform's authentication requirements and policies
